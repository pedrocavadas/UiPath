#ReFramework Without Queue

##Dispatcher
In the Config.xlsx file, inside the "Data" folder of the project, in the "Value" column:
- After the "StartFilePath" field, write the path to the spreadsheet with the initial inputs.
- After the "ExcelFilePath" field, write the path to the Custom Excel spreadsheet.

##Performer
In the Config.xlsx file, inside the "Data" folder of the project, in the "Value" column:
- After the "UiDemoCredentials" field, write the name of the asset that holds the UiDemo credentials in your Orchestrator.
- After the "ExcelFilePath" field, write the path to the Custom Excel spreadsheet.
